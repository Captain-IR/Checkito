### ✨ [Demo](https://todolisa-web.netlify.app/)

## Prerequisites

- node >= 10.18.1
- npm >= 6.13.4
- yarn >= 1.21.1

## Install

```sh
yarn
```

## Usage

```sh
quasar dev
```

## Build the app for production

```bash
quasar build
```
